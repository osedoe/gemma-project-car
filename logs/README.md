# Work Logs

Chronological record of all work performed on Gemma.

## Naming Convention

Log files should be named: `YYYY-MM-DD-brief-description.md`

Examples:
- `2025-01-15-engine-disassembly.md`
- `2025-02-03-rust-repair-passenger-door.md`
- `2025-03-20-brake-system-rebuild.md`

## Log Template

When creating a new log entry, copy this template:

```markdown
# [Work Description] - YYYY-MM-DD

**Date:** YYYY-MM-DD
**Duration:** X hours
**Location:** [Garage / Shop / Driveway]
**Weather:** [If working outdoors]

## Work Performed

[Detailed description of work completed]

## Parts Used

- [Part 1]
- [Part 2]

## Tools Used

- [Tool 1]
- [Tool 2]

## Issues Encountered

[Any problems, surprises, or complications]

## Next Steps

[What needs to be done next]

## Photos

[References to photos in ../photos/]

## Notes

[Any additional observations or lessons learned]
```

## Quick Reference

To create a new log:
1. Copy the template above
2. Create a new file with the naming convention
3. Fill in all sections
4. Update current-tasks.md if milestones were reached
