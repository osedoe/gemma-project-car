# Project Milestones

Major phases and milestones for Gemma's restoration.

## Milestone Phases

### Phase 1: Initial Assessment & Planning
- [x] Receive car and assess condition
- [x] Document initial condition (photos recommended)
- [ ] Create comprehensive parts list
- [ ] Establish budget
- [ ] Set timeline

### Phase 2: Structural & Rust Repair
- [x] Remove interior components
- [x] Cut out rusty floor sections (rear)
- [x] Fabricate and weld floor patches
- [x] Apply weld-through primer to patches
- [ ] Install Reddish Motorsport v3 subframe reinforcement plates
- [ ] Complete any remaining body rust repairs
- [ ] Assess and repair front floor if needed

### Phase 3: Undercarriage Restoration
- [ ] Clean entire underside thoroughly
- [ ] Strip old paint and undercoating
- [ ] Treat any remaining surface rust
- [ ] Prime undercarriage
- [ ] Paint undercarriage with durable coating
- [ ] Allow proper cure time

### Phase 4: Rear Suspension & Brake System
- [ ] Remove old rusty brake brackets
- [ ] Remove old fuel tank brackets
- [ ] Install new brake brackets
- [ ] Install new fuel tank brackets
- [ ] Inspect/replace rear subframe
- [ ] Install/rebuild rear control arms
- [ ] Install/rebuild rear trailing arms
- [ ] Install rear suspension components (shocks, springs)
- [ ] Install rear brake calipers and rotors
- [ ] Install new brake lines
- [ ] Bleed brake system

### Phase 5: Electrical System Restoration
- [ ] Inspect wiring harness for damage
- [ ] Clean all connectors
- [ ] Reinstall main wiring harness
- [ ] Connect all rear electrical components
- [ ] Test lighting circuits
- [ ] Test sensor circuits
- [ ] Verify ground connections

### Phase 6: Engine Revival (First Start)
- [ ] Change all fluids (oil, coolant, transmission, diff)
- [ ] Install new battery
- [ ] Replace fuel filter
- [ ] Replace air filter
- [ ] Replace spark plugs
- [ ] Inspect/replace ignition coils if needed
- [ ] Check fuel delivery system
- [ ] Prime oil system
- [ ] Attempt first start
- [ ] Diagnose and fix starting issues

### Phase 7: Front Suspension & Drivetrain
- [ ] Inspect front suspension components
- [ ] Replace worn bushings
- [ ] Install/rebuild front brake system
- [ ] Check steering components
- [ ] Inspect drive axles and CV boots
- [ ] Service differential
- [ ] Inspect transmission (if issues found)

### Phase 8: Interior & Exterior Finishing
- [ ] Reinstall interior panels
- [ ] Install seats
- [ ] Install trim pieces
- [ ] Detail engine bay
- [ ] Polish/detail exterior
- [ ] Address any remaining cosmetic issues

### Phase 9: Testing & Shakedown
- [ ] Initial test drives
- [ ] Suspension alignment
- [ ] Brake system testing
- [ ] Engine performance check
- [ ] Address any issues found
- [ ] Final road test

### Phase 10: OEM Restoration Complete
- [ ] All systems operational
- [ ] Documentation complete
- [ ] First proper drive
- [ ] Enjoy Gemma!

---

## Notes

Adjust these phases based on your specific restoration needs. Add dates when milestones are reached.
