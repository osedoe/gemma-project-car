# Parts Inventory

This directory tracks all parts purchased, needed, and installed for Gemma's restoration.

## Files

- `purchased.md` - Parts that have been purchased
- `needed.md` - Parts still needed (wish list)

## Adding a Part

When adding a part to `purchased.md`, include:
- Part name and description
- Part number (if applicable)
- Supplier/vendor
- Purchase date
- Cost
- Status (ordered, received, installed)
- Notes (condition, fitment issues, etc.)
